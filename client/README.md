# Luxe India — Premium E-Commerce

India's premium destination for curated fashion, electronics, and lifestyle products.

## Project Structure

This project is built with:
- **Vite**
- **TypeScript**
- **React**
- **shadcn-ui**
- **Tailwind CSS**

## Getting Started

1.  **Clone the repository.**
2.  **Install dependencies:**
    ```sh
    npm install
    ```
3.  **Start the development server:**
    ```sh
    npm run dev
    ```

## Deployment

The application can be built for production using:
```sh
npm run build
```
The resulting build directory (`dist`) can be deployed to any static host.
